# 📝 Notes AI App

A modern cross-platform Notes application built using **Expo** and **React Native**, designed with a clean UI and scalable architecture. This project is structured to support future AI-powered features like smart summaries and intelligent organization.

---

## 🚀 Features

- 🗒️ Create and manage notes
- 🎨 Clean and minimal UI
- 📱 Cross-platform (Android & iOS)
- ⚡ Fast and lightweight performance
- 🧱 Scalable folder structure

---

## 🛠 Tech Stack

- Expo
- React Native
- TypeScript
- AsyncStorage (for local data)
